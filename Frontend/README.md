# Frontend Dashboard

A responsive dashboard UI for the Sciqus Frontend Task. Built with React, TypeScript, and Tailwind CSS.

**Live Demo:** https://aakifmudelfrontendtask.vercel.app/

## What's Inside

- Responsive layout (desktop sidebar, mobile drawer)
- Image carousel with auto-rotation
- Interactive slider widgets
- Static content cards
- Toggle buttons with state management

## Stack

- React 19
- TypeScript
- Tailwind CSS
- Vite

## Run It

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Build

```bash
npm run build
```
